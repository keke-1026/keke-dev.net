RSA暗号の使い方と仕組み

各値の意味

p, q
2つの異なる素数です。すべての計算のベースになります。
n
p と q を掛け合わせた値です (n = p × q)。暗号化と復号の両方で使われます。
φ(n)
(p-1) と (q-1) を掛けた値です。鍵を計算するためだけに使われ、外部には公開しません。
e
公開鍵の一部です。φ(n) と互いに素（最大公約数が1）になるように選びます。
d
秘密鍵です。「e × d を φ(n) で割った余りが 1」になるように計算された特別な値です。

暗号化の仕組み

元の文字を数値(M)に変換し、以下の計算を行います。

暗号文(C) = Me mod n

「Mをe乗して、nで割った余り」が暗号文になります。公開鍵(e, n)を知っていれば誰でも暗号化できます。

復号（解読）の仕組み

暗号文の数値(C)を元に戻すには、以下の計算を行います。

元の文字(M) = Cd mod n

「Cをd乗して、nで割った余り」を計算すると、魔法のように元の数値Mに戻ります。これは秘密鍵(d)を知っている人にしかできません。
